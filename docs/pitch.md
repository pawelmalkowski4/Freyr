# Pitch Notes

## One-liner

Freyr's Eye to AIoT strażnik upraw: czujnik BLE + aplikacja mobilna, która rozpoznaje roślinę, mierzy warunki i tłumaczy dane na konkretne działania.

## Demo flow

1. Użytkownik dodaje roślinę ze zdjęcia.
2. Aplikacja rozpoznaje gatunek i zapisuje zakresy optymalne.
3. BLE dostarcza bieżące odczyty.
4. Dashboard pokazuje stan i priorytet działań.
5. Chat tłumaczy, co zrobić dalej.

## Ryzyka demo

- Brak gotowego firmware.
- Brak wdrożonego backendu proxy.
- Obecnie część danych w UI jest mockowana.
